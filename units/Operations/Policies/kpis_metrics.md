# Operations KPIs and Metrics

## Overview

Operational performance indicators measuring efficiency, quality, and service delivery excellence.

## Operational KPIs

| KPI | Target | Frequency | Owner |
|-----|--------|-----------|-------|
| OEE (Overall Equipment Effectiveness) | 85%+ | Daily | Operations Mgr |
| On-Time Delivery | 98%+ | Weekly | Logistics |
| First Pass Yield | 99%+ | Daily | Quality |
| Cost per Unit | Trending down | Monthly | Finance/Ops |
| Capacity Utilization | 80-90% | Weekly | Planning |
| Order Accuracy | 99.5%+ | Weekly | Fulfillment |
| Cycle Time | Per process SLA | Daily | Process Owner |

## Efficiency Metrics

### Process Efficiency

- Throughput rate
- Work-in-progress (WIP) levels
- Bottleneck identification
- Value stream mapping scores

### Resource Utilization

- Labor efficiency: 90%+
- Equipment utilization
- Space utilization
- Energy consumption per unit

### Productivity

- Units per labor hour
- Output per shift
- Overtime percentage: <10%

## Quality Metrics

| Metric | Target | Action Threshold |
|--------|--------|------------------|
| Defect Rate | <0.1% | >0.5% triggers review |
| Customer Complaints | <1 per 1000 orders | >3 triggers escalation |
| Returns Rate | <2% | >5% triggers investigation |
| Inspection Pass Rate | 99%+ | <95% triggers hold |

## Continuous Improvement

### Lean Metrics

- 5S audit scores: 90%+
- Kaizen events completed
- Waste reduction %
- Standard work adherence

### Six Sigma

- DPMO (Defects Per Million Opportunities)
- Process capability (Cp, Cpk)
- Control chart stability

## Dashboard

| Dashboard | Content | Refresh |
|-----------|---------|---------|
| Operations Floor Dashboard | Real-time production metrics | Live |
| Executive Operations | Summary KPIs, trends | Daily |
| Quality Dashboard | Defects, yields, SPC charts | Real-time |
| Supply Chain Dashboard | Inventory, deliveries | Daily |
