# Education (L&D) KPIs and Metrics

## Overview

Learning and development performance indicators measuring training effectiveness, skill development, and organizational capability.

## Education KPIs

| KPI | Target | Frequency | Owner |
|-----|--------|-----------|-------|
| Training Hours per Employee | 40+ hours/year | Quarterly | L&D Director |
| Training Completion Rate | 95%+ required | Monthly | L&D Manager |
| Learning Satisfaction | 90%+ | Per program | Program Lead |
| Skill Gap Closure | Tracked | Quarterly | L&D Director |
| Time to Proficiency | Decreasing | Per role | L&D Manager |
| Training ROI | Documented | Annual | L&D Director |
| Certification Achievement | Per plan | Quarterly | Program Lead |

## Learning Delivery Metrics

### Program Metrics

| Delivery Mode | Completion Rate | Satisfaction |
|---------------|-----------------|--------------|
| Instructor-led | Track | Target 90%+ |
| E-learning | Track | Target 85%+ |
| Virtual | Track | Target 88%+ |
| On-the-job | Track | Target 92%+ |

### Content Performance

- Course completion rates
- Module-level engagement
- Assessment pass rates
- Content freshness (update cycle)

## Kirkpatrick Evaluation

### Level 1: Reaction

- Participant satisfaction: 90%+
- Net Promoter Score: Positive
- Relevance rating: High

### Level 2: Learning

- Pre/post knowledge gain
- Skill assessment improvement
- Certification pass rate

### Level 3: Behavior

- On-the-job application rate
- Manager observation scores
- 90-day behavior change

### Level 4: Results

- Business impact metrics
- Performance improvement
- ROI calculation
