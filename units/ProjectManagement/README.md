# Project Management Unit

## Overview

The Project Management Unit (PMO) drives successful project delivery through standardized methodologies and governance.

## Mission

To enable organizational success by delivering projects on time, within budget, and to specification.

## Key Responsibilities

- Project portfolio management
- Resource capacity planning
- Project methodology standards
- Risk and issue management
- Project status reporting
