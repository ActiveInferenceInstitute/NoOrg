# Project Management KPIs and Metrics

## Overview

PMO performance indicators measuring project delivery success, resource efficiency, and portfolio health.

## Project Management KPIs

| KPI | Target | Frequency | Owner |
|-----|--------|-----------|-------|
| On-Time Delivery | 90%+ | Monthly | PMO Director |
| On-Budget Delivery | 90%+ | Monthly | PMO Director |
| Scope Adherence | 95%+ | Per project | PM |
| Resource Utilization | 80-85% | Weekly | Resource Mgr |
| Customer Satisfaction | 90%+ | Per project | PM |
| Active Risk Mitigation | 100% documented | Weekly | PM |
| Lessons Learned Capture | 100% of projects | Per project | PMO |

## Portfolio Health Metrics

### Pipeline and Capacity

- Projects in flight vs. capacity
- Upcoming project demand
- Resource bottlenecks
- Skills gap analysis

### Status Distribution

| Status | Target % | Action |
|--------|----------|--------|
| Green | 75%+ | Maintain |
| Yellow | <20% | Monitor |
| Red | <5% | Escalate |

## Delivery Metrics

### Schedule Performance

- Schedule Performance Index (SPI): ≥1.0
- Milestone achievement rate
- Schedule variance trending
- Critical path adherence

### Cost Performance

- Cost Performance Index (CPI): ≥1.0
- Earned Value metrics (EV, AC, PV)
- Estimate to Complete accuracy
- Change order impact

### Quality

- Defects found in delivery
- Rework percentage: <10%
- Acceptance criteria met: 100%
- Post-implementation issues

## Resource Management

| Metric | Target | Tracking |
|--------|--------|----------|
| Billable utilization | 80%+ | Weekly |
| Bench time | <10% | Weekly |
| Overtime | <5% | Monthly |
| Contractor ratio | Per policy | Monthly |

## Process Compliance

### Methodology Adherence

- Stage-gate compliance: 100%
- Required artifacts complete: 100%
- PM tool usage: 100%
- Status reporting on-time: 100%
