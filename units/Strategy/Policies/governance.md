# Strategy Governance

## Overview

Governance framework for the Strategy Unit.

## Decision Rights

| Decision Type | Authority Level | Approval Required |
|--------------|-----------------|-------------------|
| Strategic | Director | Executive |
| Operational | Manager | Director |
| Tactical | Specialist | Manager |

## Escalation Matrix

| Issue Severity | First Contact | Escalation Path |
|----------------|---------------|-----------------|
| Critical | Director | Executive → Board |
| High | Manager | Director → Executive |
| Medium | Specialist | Manager → Director |
| Low | Analyst | Specialist → Manager |

## Review Cadence

- Weekly: Team standup
- Monthly: Operations review
- Quarterly: Performance review
- Annual: Strategic planning
