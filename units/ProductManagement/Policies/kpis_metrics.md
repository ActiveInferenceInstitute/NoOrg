# Product Management KPIs and Metrics

## Overview

Product management performance indicators measuring product success, market fit, and delivery effectiveness.

## Product KPIs

| KPI | Target | Frequency | Owner |
|-----|--------|-----------|-------|
| Product Revenue | Per plan | Quarterly | VP Product |
| Feature Adoption Rate | 60%+ in 90 days | Per release | PM |
| Customer Satisfaction (Product) | 90%+ | Quarterly | PM |
| Net Promoter Score (Product) | 50+ | Quarterly | PM |
| Time to Market | Decreasing trend | Per release | PM |
| Roadmap Delivery | 90%+ as committed | Quarterly | VP Product |
| Product-Market Fit Score | 40%+ "very disappointed" | Quarterly | PM |

## Product Health Metrics

### Engagement

- Daily/Monthly Active Users (DAU/MAU)
- Session duration
- Feature usage heatmap
- Retention curves (D1, D7, D30)

### Value Delivery

- Core workflow completion rate
- Time to value for new users
- Aha moment achievement
- Expansion usage

## Roadmap Metrics

### Planning Effectiveness

| Metric | Target | Tracking |
|--------|--------|----------|
| Roadmap accuracy | 85%+ | Quarterly |
| Scope changes | <20% | Per release |
| Dependencies managed | 100% | Per release |
| Tech debt balance | Sustainable | Quarterly |

### Prioritization

- Feature requests processed
- Prioritization framework adherence
- Stakeholder alignment score
- Customer-requested vs. strategic mix

## Discovery Metrics

### Research

- Customer interviews conducted
- Usage data analysis frequency
- Experiment velocity
- Hypothesis validation rate

### Validation

- Prototypes tested
- A/B tests run
- Win rate on experiments
- Failed fast (good failures)

## Delivery Metrics

### Release Quality

| Metric | Target | Tracking |
|--------|--------|----------|
| Release frequency | Per cadence | Weekly |
| Bugs escaped | <5% | Per release |
| Hotfix rate | <10% of releases | Monthly |
| Rollback rate | <5% | Per release |

## Business Impact

- Revenue per feature/product
- Customer acquisition from product
- Churn reduction from features
- Competitive differentiation
