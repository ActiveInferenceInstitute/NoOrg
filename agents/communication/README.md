# Agent Communication Protocols

## Overview

This directory contains documentation for agent communication patterns, message formats, and interaction protocols used in the NoOrg multi-agent framework.

## Contents

| File | Description |
|------|-------------|
| `interaction-patterns.md` | Common patterns for agent-to-agent interactions |
| `message-formats.md` | Standardized message format specifications |
| `protocols.md` | Communication protocol definitions |

## Purpose

These documents define how agents communicate with each other in the system:

- **Message Structure**: Standard formats for inter-agent messages
- **Interaction Patterns**: Request-response, pub-sub, broadcast patterns
- **Protocol Specifications**: Handshake, acknowledgment, and error handling
