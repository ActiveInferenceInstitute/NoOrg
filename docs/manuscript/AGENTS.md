# NoOrg manuscript guidance

The manuscript source is a maintained publication artifact. Chapters are numbered so the rendering order is deterministic. All changing values must enter through `{{TOKEN}}` variables generated by `scripts/manuscript.ts`; prose must not contain hand-maintained measurements.

Source figures live in `figures/` as Mermaid documents. Rendered figures, hydrated chapters, evidence, HTML, PDF, and checksums belong under `output/`, which is generated and ignored.

Use Pandoc-crossref labels for equations, figures, tables, and sections. Every reference must resolve during `npm run manuscript:validate`. Bibliographic keys must exist in `references.bib`.
