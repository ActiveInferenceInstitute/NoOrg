# Glossary {#sec:glossary}

**Agent.** An initialized executable object with a descriptor, input schema, result schema, and lifecycle methods.

**Capability.** A named property used by the registry to match a task to an agent.

**Lease.** The reservation returned by the registry while an agent execution is active.

**Provider.** A dependency that returns schema-validated structured data and usage metadata.

**Repository.** The authority that validates and persists task state transitions.

**State revision.** A monotonically increasing number attached to a serialized state envelope.

**Terminal task.** A task in completed, failed, or cancelled status.
