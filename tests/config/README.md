# Test Configuration

Configuration files for the test suite.

## Overview

This directory contains configuration files for:
- Jest (TypeScript/JavaScript tests)
- pytest (Python tests)
- TypeScript compilation

## Files

### jest.setup.ts

Jest setup file for TypeScript tests.

### pytest.ini

pytest configuration for Python tests.

### tsconfig.json

TypeScript configuration for test compilation.
