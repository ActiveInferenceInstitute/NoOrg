# Tests/Tests - Technical Documentation

## Overview

This directory contains additional test organization for the NoOrg framework. This structure provides a nested test hierarchy.

## Directory Structure

```text
tests/
├── __init__.py
├── integration/    # Integration tests
├── system/         # System tests
└── unit/           # Unit tests
```text

## Note

This directory exists as an alternate test organization structure. Primary tests are in the parent `tests/` directory.
