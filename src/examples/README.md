# Source Examples

Example implementations demonstrating framework usage.

## Overview

This directory contains example code showing how to use the NoOrg Multi-Agent Framework.

## Examples

Examples demonstrate:
- Basic agent coordination
- Complex multi-agent workflows
- Integration patterns
- Organizational unit workflows

## Usage

Examples can be run directly or imported as modules:

```typescript
import { runExample } from './example-file';
await runExample();
```
