## Components

### 1. Core Workflow (`calm_tech_workflow.ts`)
**Primary orchestration system demonstrating all 8 calm technology principles**

- **Attention-Aware Agent**: Monitors user attention state and adapts behavior
- **Ambient Notification System**: Communicates without demanding attention
- **Resilient Calm System**: Maintains functionality during failures
- **Cognitive Load Assessor**: Measures and optimizes attention demands

### 2. Ambient Notifications (`ambient_notifications.ts`)
**Sophisticated notification system respecting attention and cognitive load**

- Multi-modal ambient feedback (visual, auditory, tactile, spatial)
- Context-aware notification adaptation
- Peripheral attention respect
- Rule-based notification management

### 3. Peripheral Awareness (`peripheral_awareness.ts`)
**Dynamic interface adaptation based on attention state and context**

- Attention state transition management
- Context-aware interface morphing
- Peripheral information display strategies
- Interface zone organization

### 4. Resilient Design (`resilient_design.ts`)
**Graceful degradation patterns for system failures**

- Fallback mode management
- System recovery mechanisms
- Calm failure handling
- Progressive enhancement patterns

## Key Features

### Attention State Management
```typescript
// Automatic attention state assessment and adaptation
await attentionAgent.assessAttentionState();
await attentionAgent.adaptToAttentionState();
```

### Ambient Communication
```typescript
// Context-aware notifications that don't demand attention
await notificationEngine.sendAmbientNotification(
  'Meeting in 15 minutes',
  NotificationUrgency.NORMAL,
  { userAttention: AttentionState.PERIPHERAL }
);
```

### Graceful Degradation
```typescript
// System continues functioning even during failures
await resilientManager.handleDegradation(
  DegradationTrigger.CONNECTIVITY_LOSS,
  'major'
);
```

### Cognitive Load Optimization
```typescript
// Automatic assessment and optimization of attention demands
await loadAssessor.assessCognitiveLoad('dashboard_view');
await loadAssessor.optimizeForLoad('dashboard_view');
```

## Implementation Patterns

### 1. Attention-First Design
```typescript
// Always consider attention state before presenting information
const currentAttention = await attentionAgent.assessAttentionState();
const adaptedContent = adaptForAttention(content, currentAttention);
```

### 2. Ambient Information Architecture
```typescript
// Structure information for peripheral consumption
interface AmbientInfo {
  content: string;
  urgency: NotificationUrgency;
  zone: AmbientZone;
  persistence: PersistenceLevel;
}
```

### 3. Graceful Degradation Strategy
```typescript
// Plan for failures from the beginning
const fallbackModes = [
  'offline_queue',      // When connectivity fails
  'essential_only',     // When resources are limited
  'ambient_indicators'  // When interaction fails
];
```
