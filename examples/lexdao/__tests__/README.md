# LexDAO Tests - README

## Overview

Test suites for the LexDAO example implementation using Jest.

## Test Files

- `agents.test.ts` - Tests for LexDAO agent functionality
- `workflow.test.ts` - Tests for workflow execution
- `setup.ts` - Test configuration and setup

## Usage

```bash
# Run all tests
npm test

# Run with coverage
npm test -- --coverage

# Run specific test file
npm test -- agents.test.ts
```text
