# Scripts

Utility scripts for development, maintenance, and analysis.

## Overview

This directory contains utility scripts for:
- Performance analysis and optimization
- Security scanning
- Documentation generation
- Codebase analysis

## Scripts

### advanced-performance-analyzer.js

Advanced performance analysis and optimization.

### advanced-security-scanner.js

Comprehensive security scanning and vulnerability detection.

### generate-docs-structure.js

Documentation structure generation.

### performance-optimizer.js

Performance optimization utilities.

### security-scanner.js

Security scanning utilities.

## Usage

Scripts can be run directly:

```bash
node scripts/performance-optimizer.js
```text
